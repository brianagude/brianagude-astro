Let's Be Productive! is a task manager I built for how my brain works.<br/><br/>Every app I've tried had too much setup or too many features that I never used, so I made something minimal. It's got one place for tasks, tags to sort by type, a stats page to see where my time actually goes, and a focus timer for deep work. + everything saves to local storage.
